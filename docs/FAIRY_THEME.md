## 1. 产品定位

| 项 | 内容 |
|----|------|
| 路径 | `/fairy` |
| Hub 标签 | Fairy · **Medium** |
| 气质 | 手绘卡通、偏女性向、柔和配色、优雅可爱 |
| 总关数 | **85**（按中等模板：15 + 40 + 10 + 20） |
| 限时 | 每关 10s |

一句话：在一群仙女里，找出那个「不太一样」的；前期靠造型差，中期靠部件差，后期靠 5×5 密网格 + 换位加压（部件 twin 仍须缩略可辨）。

## 2. 关卡曲线（锁定）

| 段 | 关号 | 关数 | 玩家感受 | 差异规则 | 网格 | FX |
|----|------|------|----------|----------|------|-----|
| 简单 | 1–15 | 15 | 轻松 | Odd = **明显不同的仙女**（剪影或主色一眼可辨） | 3×3→4×4 | 无 |
| 中等 | 16–55 | 40 | 主力 | **同一基础造型**，仅 **一个部位** 不同：发色 / 衣服 / 道具 / 装饰 | 4×4 | 无 |
| 过渡 | 56–65 | 10 | 略轻松 | 差异略放大（仍属部件级，但比中等段更好认），**不上换位** | 4×4 | 无 |
| 高难 | 66–85 | 20 | 难 | **清晰部件 twin**（5×5 更密）+ 图片可移动（`teleport`）；原 micro 资产暂留不用 | 5×5 | 建议 71+ 开 teleport |

区间说明（若以后要加减关）：简单 10–20、中等 30–50、过渡固定 10、高难固定 20；改关数时保持四段比例。

## 3. 资产规划（出图前）

### 3.1 规格

- **1024×1024** PNG，白底，单角色居中，留边 10–12%  
- 风格锚点与提示词：待补 `docs/FAIRY_PROMPTS.md`（开工出图时写）

### 3.2 建议角色卡（草案，出图时可微调）

主阵容约 **8 只**基础仙女（剪影可分），供简单段「明显不一样」轮换：

| ID | 辨识剪影（草案） |
|----|------------------|
| F01 | 长直发 + 小魔杖 |
| F02 | 双马尾 + 星星裙 |
| F03 | 短发 + 大蝴蝶结 |
| F04 | 花冠 + 蓬蓬裙 |
| F05 | 螺旋卷发 + 翅膀偏大 |
| F06 | 帽檐仙女 |
| F07 | 铃兰主题配色 |
| F08 | 月光银白主题 |

### 3.3 中等段 Twin 类型（必须单独出图）

对同一 `F0x` 做变体，例如：

| 后缀语义 | 差异部位 |
|----------|----------|
| `_hair` | 发色（金 ↔ 粉 ↔ 蓝等） |
| `_dress` | 衣服主色 / 款式一点 |
| `_prop` | 手持道具（魔杖 ↔ 花朵 ↔ 灯笼） |
| `_deco` | 装饰（耳环、项链、翅膀尖端） |

高难段原计划用「更小」twin（鞋扣 / 裙星 / 结心宝石等），实测在 5×5 缩略下几乎不可见，已改用与中等段相同的高对比部件 twin；micro 文件保留待重画。
