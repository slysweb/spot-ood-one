## 1. 产品定位

| 项 | 内容 |
|----|------|
| 路径 | `/food` |
| Hub 标签 | Foods · **Hard** |
| 气质 | 卡哇伊卡通食物；同构图 twin，仅差一个小部件 |
| 总关数 | **20**（困难短包：5 + 9 + 6） |
| 限时 | 每关 10s |
| 美术 | **1024×1024 WebP**，白底，单食物居中 |

一句话：每一关一种食物；一堆几乎一样的图里，找出那个「多了 / 少了一个小配料」的（如披萨有无草莓）。

---

## 2. 关卡曲线（锁定 · 20 关）

| 段 | 关号 | 关数 | 玩家感受 | 差异规则 | 网格 | FX |
|----|------|------|----------|----------|------|-----|
| 热身 | 1–5 | 5 | 仍可读 | 同食物 twin：部件差稍明显 | 4×4 | 无 |
| 主体 | 6–14 | 9 | 难 | 同食物微差 twin；网格加密 | 4×4→5×5 | 无 |
| 压迫 | 15–20 | 6 | 更难 | 最近 twin + 5×5；可换位 | 5×5 | teleport（17+） |

**强制规则：**

- **每关食物不重复**（20 关 = 20 种食物）。  
- Odd 与 Base 必须是 **同一食物的 twin 对**，禁止跨食物「藏一只不同种类」。  
- 差异靠 **画出来的部件**（草莓、樱桃、吸管、芝麻等），禁止只靠 CSS 微滤镜。
