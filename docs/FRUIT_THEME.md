## 1. 产品定位

| 项 | 内容 |
|----|------|
| 路径 | `/fruit` |
| Hub 标签 | Fruits · **Medium** |
| 气质 | 卡通可爱（kawaii chibi）、圆润描边、柔和上色 |
| 总关数 | **30**（中等短包：8 + 12 + 4 + 6） |
| 限时 | 每关 10s |

一句话：在一群水果里，找出那个「类别或形状不太一样」的；前期跨品类，中期易错认近亲对，后期密网格 + 换位。

## 2. 关卡曲线（锁定 · 30 关）

| 段 | 关号 | 关数 | 玩家感受 | 差异规则 | 网格 | FX |
|----|------|------|----------|----------|------|-----|

## 3. 资产规划

### 3.1 规格

- **1024×1024** WebP，白底，单果居中，留边约 10–12%
- 风格：卡通可爱、粗描边、赛璐璐着色（见 `docs/FRUIT_PROMPTS.md`）

### 3.2 角色卡（主阵容）

| ID | 水果 | 剪影 / 类别备注 |
|----|------|-----------------|
| R01 | 红苹果 | 圆 + 小叶柄 |
| R02 | 青苹果 | 同形不同色（易错） |
| R03 | 梨 | 上窄下宽 |
| R04 | 桃 | 心形凹口 |
| R05 | 橙 | 圆 + 脐点 |
| R06 | 橘 | 更扁圆、略小（易错） |
| R07 | 柠檬 | 椭圆两端尖 |
| R08 | 青柠 | 同形更圆/更绿（易错） |
| R09 | 香蕉 | 弯月剪影 |
| R10 | 草莓 | 心锥 + 籽点 |
| R11 | 樱桃 | 双圆 + 梗 |
| R12 | 葡萄串 | 多圆簇 |
| R13 | 西瓜片 | 三角月牙 |
| R14 | 菠萝 | 椭圆 + 冠叶 |
| R15 | 奇异果剖面 | 椭圆褐皮绿心 |
| R16 | 芒果 | 肾形 |

### 3.3 易错认对（中等/高难主力）

| Base | Odd | 差异重心 |
|------|-----|----------|
| R01 | R02 | 同形不同色（红/青苹） |
| R05 | R06 | 橙 vs 橘（圆度/大小感） |
| R07 | R08 | 柠檬 vs 青柠 |
| R01 | R04 | 苹果 vs 桃（凹口） |
| R01 | R03 | 苹果 vs 梨（轮廓） |
| R11 | R10 | 樱桃 vs 草莓（形状） |
| R12 | R11 | 葡萄串 vs 樱桃（结构） |
